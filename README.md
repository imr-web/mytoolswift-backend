# MyToolSwift AI Backend
This is the high-performance Python engine for the **MyToolSwift** Background Remover.

## Features
- **AI Engine:** U2Net-P (Pocket) for universal object detection.
- **Support:** Perfectly isolates motorcycles, buildings, pets, and people.
- **API:** FastAPI powered for low-latency responses.

## Deployment
Hosted on **Render.com** to bypass mobile RAM limits.
